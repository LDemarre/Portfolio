INFORMACIÓN COMPLETA DEL PROYECTO

El proyecto es un tablero de madera que mide 35cm de largo, 35cm de ancho y 6cm de alto y que por debajo tiene 4 patas de goma. Dentro tiene una cuadrícula de 10 celdas x 10 celdas separadas unas de otras por madera. Cada celda mide 3,5cm x 3,5cm y contiene un LED. Este tablero cumple el rol de una pantalla, siendo las celdas, los pixeles. Estos serán controlados por una aplicación móvil que se comunicará con el tablero mediante bluetooth. Teniendo un sistema de puntaje que se mostrará en la misma aplicación y al final de cada juego en el tablero.

Primera fase de creación y armado

Explicación:

Inicialmente conseguimos una pieza de madera, la cual dividimos en una cuadricula de 10x10 celdas de 3,3cm tanto de ancho como de largo cada una. Luego colocamos las tiras de leds de forma tal que cada celda tenga un led en el centro. Por último conectamos las tiras de leds entre sí en forma de zigzag.

Problemas y soluciones:

Uno de los posibles problemas que pudo haber ocurrido sería una mala colocación de los leds, resultando en que las líneas queden torcidas, aunque esto no sucedió y aunque hubiera sucedido, no presentaría un gran problema en el proyecto final.

    Explicación:

    Teniendo ya la base con los leds armada, conseguimos 18 tablas de madera a las que le hicimos cortes para poder ensamblarlas entre sí. Estas tablas ensambladas darían como resultado la cuadrícula que se encarga de delimitar cada celda.

    Problemas y soluciones:

    Los problemas que podríamos haber tenido están relacionados con los cortes de las maderas. Estos podrían haber sido demasiado profundos, haciendo que las maderas quedasen flojas y el ensamble no quede firme. O podrían haber quedado demasiado poco profundos, resultando en que las maderas no puedan encajarse lo suficiente. Para el primer problema la única solución posible sería empezar desde cero con otra madera, mientras que para el segundo, simplemente bastaría con hacer un poco más profundo el corte y fijarse nuevamente si las maderas se pueden ensamblar correctamente.

En estas realizamos un corte a pocos milímetros de la parte superior de una de sus caras para poder encajar una lámina de vidrio opaco, la cual además usa de apoyo la cuadrícula de madera. Este vidrio opaco se encarga de que la luz de los LEDs se vea en toda la zona que cubre la celda para dar la impresión de ser un "Pixel grande", en vez de iluminarse solo un pequeño punto en el medio de la celda, como lo haría sin este.

Un corte poco profundo significaría que el vidrio no entraría en su lugar y uno demasiado profundo le impediría al vidrio estar fijo y este se movería libremente.

    Explicación:

    En nuestro circuito utilizamos una fuente de 5V y 0.7A que alimenta al Arduino NANO y a la tira de LEDs a través de un capacitor que funciona como un filtro para evitar los picos de tensión que se produzcan nada más conectar el tablero. La razón por la que los LEDs son alimentados directamente de la fuente en lugar desde el Arduino es porque este no suministra sufiente corriente a los leds para lograr prender varios a la vez. Esta tira de LEDs además está conectada al pin D6 del Arduino a través de una resistencia, que se va a encargar de darle la información necesaria para definir cómo y que LEDs deberán encenderse. También utilizamos un módulo bluetooth que es el que se va a encargar de comunicar la aplicación móvil con el tablero. Este tiene sus pines de transmisión y recepción de datos conectados a los del Arduino. El pin TX del módulo bluetooth está conectado al RX del Arduino de forma directa, mientras que la otra conexión está hecha mediante un divisor de tensión. La razón de esto es debido al voltaje que envian los pines a la hora de transmitir/recibir datos. El Arduino lo hace en 5V, mientras que el módulo bluetooth lo hace en 3.3V, si no se utilizara el divisor de tensión, el módulo recibiría una tensión mayor a la soportada, lo que causaría que este se queme. Mientras que los 3.3V del módulo bluetooth sí son recibidos como un 1 lógico por el Arduino, por lo que por esa parte no habría problemas. Por último, el módulo bluetooth está siendo alimentado por el Arduino a través del pin de 5V.

        /* Librería */
    #include <Adafruit_NeoPixel.h>
    
    /* Variables y constantes del tablero */
    #define FIELD_WIDTH   10
    #define FIELD_HEIGHT  10
    const int NUMPIXELS = FIELD_WIDTH * FIELD_HEIGHT;
    const int PIN = 6;
    int tx = 1;
    int rx = 0;
    String inputString = "";
    byte BALL_DIR;

    void readInput() {
    if (Serial.available()) {
        while (Serial.available()) {
            char inChar = Serial.read();
            inputString += inChar;
        }
        if (inputString == "u") {
            curControl = BTN_UP;
        }
        else if (inputString == "d") {
            curControl = BTN_DOWN;
        }
        else if (inputString == "l") {
            curControl = BTN_LEFT;
        }
        else if (inputString == "r") {
            curControl = BTN_RIGHT;
        }
        else if (inputString == "e") {
            curControl = BTN_EXIT;
        }
        else if (inputString == "s") {
            curControl = BTN_START;
        }
        else if (inputString == "c") {
            curControl = BTN_CENTER;
        }
        else if (inputString == "f") {
            curControl = BTN_FIRE;
        }
        else if (inputString == "1") {
            curControl = BTN_SNAKE;
        }
        else if (inputString == "2") {
            curControl = BTN_PONG;
        }
        else if (inputString == "3") {
            curControl = BTN_BREAKOUT;
        }
        else if (inputString == "4") {
            curControl = BTN_SPACE;
        }
        else if (inputString == "5") {
            curControl = BTN_DICE;
        }
        inputString = "";
      }
    }


    /* Pintado de pixeles */
    int pos = 0;//Posicion real en numeros de pixeles

    void setTablePixel(int x, int y, int r, int g, int b) {
        if (y % 2 == 0) {
            pos = y * FIELD_WIDTH + x;
        } else {
            pos = y * FIELD_WIDTH + ((FIELD_WIDTH - 1) - x);
        }
        pixels.setPixelColor(pos, pixels.Color(r, g, b));
    }

    void clearTablePixels() {
        for (byte i = 0; i < NUMPIXELS; i++) {
            pixels.setPixelColor(i, pixels.Color(0, 0, 0));
        }
    }

    /* Detección de colisiones */
    boolean collide(byte x1, byte x2, byte y1, byte y2) {
        if ((x1 == x2) && (y1 == y2)) {
            return true;
        }
        else {
            return false;
        }
    }

    void start () {
    clearTablePixels();
    pixels.show();
    boolean inicio = true;

    while (inicio) {
        readInput();

        if (curControl == BTN_START) {
            inicio = false;
            mainMenu();
            break;
        }
      }
    }

    void mainMenu() {
    curControl = BTN_NONE;
    boolean menu = true;

    while (menu) {
        unsigned long prevUpdateTime = 0;
        unsigned long curTime;

        for (byte i = 0; i < NUMPIXELS; i++) {
            byte rr = random(15);
            byte gg = random(15);
            byte bb = random(15);
            pixels.setPixelColor(i, pixels.Color(rr, gg, bb));
            delay(10);
        }
        pixels.show();

        do {
            readInput();
            if (curControl == BTN_NONE) {
                break;
            }
            else if (curControl == BTN_SNAKE) {
                clearTablePixels();
                runSnake();
                curControl = BTN_NONE;
                menu = false;
                break;
            }
            else if (curControl == BTN_PONG) {
                clearTablePixels();
                runPong();
                curControl = BTN_NONE;
                menu = false;
                break;
            }
            else if (curControl == BTN_BREAKOUT) {
                clearTablePixels();
                runBreakout();
                curControl = BTN_NONE;
                menu = false;
                break;
            }
            else if (curControl == BTN_DICE) {
                clearTablePixels();
                runDice();
                curControl = BTN_NONE;
                menu = false;
                break;
            }
            else if (curControl == BTN_SPACE) {
                clearTablePixels();
                runSpaceInvader();
                curControl = BTN_NONE;
                menu = false;
                break;
            }
            curTime = millis();
        }
        while ((curTime - prevUpdateTime) < 250);
        prevUpdateTime = curTime;
      }
    }

    /* Funciones iniciales */
    void setup() {
        Serial.begin(9600);
        randomSeed(analogRead(0));
        pinMode(tx, OUTPUT);
        pinMode(rx, INPUT);
        pixels.begin();
        pixels.show();
        start();
    }

    void loop() {}

    Snake

        /* Variables y constantes iniciales del juego */
    byte curLength;//Longitud de la serpiente
    short xs[100];//Matriz que contiene todos los segmentos de la serpiente, 
                  //su máxima longitud es el tamaño de la matriz
    short ys[100];//xs es su posición en x, ys es su posición en y
    byte dir;//Dirección actual de la serpiente
    byte ax = 0;//Posición en x de la manzana
    byte ay = 0;//Posición en y de la manzana
    byte scoreSnake;
    byte prevScoreSnake;
    byte speedSnake;
        
    boolean snakeGameOver;
        
    /* Morir */
    void die() {
        snakeGameOver = true;
    }
        
    /* Direcciones */
    void setDirection() {
      switch (curControl) {
        case BTN_LEFT:
            if (dir == DIR_RIGHT) {
                break;
            } else {
                dir = DIR_LEFT;
                break;
            }
        case BTN_RIGHT:
            if (dir == DIR_LEFT) {
                break;
            } else {
                dir = DIR_RIGHT;
                break;
            }
        case BTN_DOWN:
            if (dir == DIR_UP) {
                break;
            } else {
                dir = DIR_DOWN;
                break;
            }
        case BTN_UP:
            if (dir == DIR_DOWN) {
                break;
            } else {
                dir = DIR_UP;
                break;
            }
        }
    }
        
        
    void snakeInit() {
        //Posición, dirección y variables iniciales de la serpiente
        curLength = 3;
        xs[0] = 3; xs[1] = 2; xs[2] = 1;
        ys[0] = FIELD_HEIGHT / 2; ys[1] = FIELD_HEIGHT / 2; ys[2] = FIELD_HEIGHT / 2;
        dir = DIR_RIGHT;
        //Crear manzana en una posición aleatoria
        ax = random(FIELD_WIDTH);
        ay = random(FIELD_HEIGHT);
        scoreSnake = 0;
        prevScoreSnake = 0;
        speedSnake = 250;
        
        snakeGameOver = false;
    }
        
    void runSnake() {
        snakeInit();
        unsigned long prevUpdateTime = 0;
        boolean snakeRunning = true;
        while (snakeRunning) {
        //Revisar colisión de la serpiente consigo misma
        byte i = curLength - 1;
        while (i >= 4) {
            if (collide(xs[0], xs[i], ys[0], ys[i])) {
                die();
                Serial.print("e");
            }
            i = i - 1;
        }
        
        if (snakeGameOver) {
            clearTablePixels();
            pixels.show();
            snakeRunning = false;
            gameOverAnim(1);
            break;
        }
        
        //Revisar colisión de la serpiente con la manzana
        if (collide(xs[0], ax, ys[0], ay)) {
            //Incrementar longitud de la serpiente
            curLength = curLength + 1;
            ++scoreSnake;
            Serial.print(scoreSnake);
            //Nuevo segmento de la serpiente con posición temporal
            xs[curLength - 1] = 255;
            ys[curLength - 1] = 255;
        
            //Crear otra manzana en una posición aleatoria
            ax = random(FIELD_WIDTH);
            ay = random(FIELD_HEIGHT);
        
            //Asegurarse de que la manzana no se genere dentro del cuerpo de la serpiente
            byte i = curLength - 1;
            for (byte i = 0; i < curLength; i++) {
                if (collide(ax, xs[i], ay, ys[i])) {
                    ax = random(FIELD_WIDTH);
                    ay = random(FIELD_HEIGHT);
                    i = 0;
                }
            }    
        }
        
        //Aumento de velocidad de la serpiente
        if ((scoreSnake - prevScoreSnake) == 5) {
            prevScoreSnake = scoreSnake;
            speedSnake = speedSnake - 25;
        }
        
        //Desfasar los segmentos del cuerpo de la serpiente
        i = curLength - 1;
        while (i >= 1) {
            xs[i] = xs[i - 1];
            ys[i] = ys[i - 1];
            i = i - 1;
        }
        
        //Determinar nueva direción de la cabeza de la serpiente
        if (dir == DIR_RIGHT) {          //Derecha
            xs[0] = xs[0] + 1;
        }
        else if (dir == DIR_LEFT) {      //Izquierda
            xs[0] = xs[0] - 1;
        }
        else if (dir == DIR_UP) {        //Arriba
            ys[0] = ys[0] - 1;
        }
        else {                           //Abajo
            ys[0] = ys[0] + 1;
        }  
        
        //Revisar si la serpiente sale de los bordes del tablero
        if ((xs[0] < 0) || (xs[0] >= FIELD_WIDTH) || (ys[0] < 0) || (ys[0] >= FIELD_HEIGHT)) {
            if (xs[0] < 0) {
                xs[0] = FIELD_WIDTH - 1;
            }
            else if (xs[0] >= FIELD_WIDTH) {
                xs[0] = 0;
            }
            else if (ys[0] < 0) {
                ys[0] = FIELD_HEIGHT - 1;
            }
            else if (ys[0] >= FIELD_HEIGHT) {
                ys[0] = 0;
            }
        }
        
        clearTablePixels();
        
        //Dibujar la manzana
        setTablePixel (ax, ay, 15, 0, 0);
        
        //Dibujar la serpiente
        for (byte i = 0; i < curLength; i++) {
            setTablePixel(xs[i], ys[i], 0, 15, 0);
        }
        
        pixels.show();
    
        //Revisar dirección de la serpiente mientras se espera otra dirección
        unsigned long curTime;
        do {
            readInput();
            if (curControl == BTN_EXIT) {
                die();
                break;
            }
            if (curControl != BTN_NONE) {
                setDirection();
            }
            curTime = millis();
        }
        while ((curTime - prevUpdateTime) < speedSnake); //Luego de que pase el tiempo equivalente a 
                                                         //"speedSnake" la serpiente se mueve
            prevUpdateTime = curTime;
      }
    }

                ++ballx;
                break;

                break;

                --bally;
                ++ballx;
                break;

                ++bally;
                --ballx;
                break;

                break;
            case BTN_RIGHT:
                break;

                if (posPlayerIzq == (FIELD_HEIGHT - 2))

                    ++posPlayerIzq;

        void runPong() {

                clearTablePixels();
                pixels.show();
                Serial.print("e");
                pongRunning = false;
                gameOverAnim(2);
                break;

            /* Revisar desde qué dirección y en qué parte del jugador 1 chocó la pelota */
            for (byte i = posPlayerIzq - 1; i <= posPlayerIzq + 1; i++) {
                if (collide(ballx, 1, bally, i)) {
                    if (i == posPlayerIzq) {
                        BALL_DIR = DIR_RIGHT;

                            BALL_DIR = DIR_UPLEFT;
                        }
                        else if (BALL_DIR == DIR_DOWNRIGHT) {
                            BALL_DIR = DIR_DOWNLEFT;
                        }
                    }
                }

    Explicación:

    Visible:

    Cuando el jugador termina un juego ya sea ganando, perdiendo o abandonando a mitad de la partida, comienza a ejecutarse esta parte del código. Al iniciarse esta parte del código, se verán parpadear los elementos en pantalla, para luego apagarse e iniciar una animación en la que la palabra "GAME OVER", pasa por la pantalla para indicar la finalización del juego. Luego, dependiendo del juego del que se haya salido, se mostrará en pantalla la puntuación que el jugador ha obtenido durante el juego. Luego de unos segundos se volverá al "Menú principal".

    No visible:

    Uno podría creer que la palabra "GAME OVER" está conformada por 8 caracteres que pasan a través de la pantalla, cuando realmente la animación está realizada cuadro por cuadro. Los números que muestran la puntuación al final de la animación del "GAME OVER", están definidos en listas al inicio del código, haciendo que el número de la izquierda y el número de la derecha sean independientes uno de otro.
