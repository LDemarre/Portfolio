PRESENTACIÓN


Materiales principales:

Módulo bluetooth HC-05

Módulo bluetooth HC-05

Arduino NANO

Arduino NANO


LEDs Direccionales WS2812B: La tira de LEDs direccionales posee cierta inteligencia que mediante algunas líneas de código nos permite encender los LEDs que deseemos con distintos colores e intensidades, dándonos mucha libertad a la hora de mostrar los juegos en el tablero.

Módulo Bluetooth HC-05: Este nos permite establecer una comunicación entre la aplicación y el tablero, pudiendo usarlo de manera inalámbrica a través de un dispositivo móvil.

Módulo Bluetooth HC-05: Este nos permite establecer una comunicación entre la aplicación y el tablero, pudiendo usarlo de manera inalámbrica a través de un dispositivo móvil.

Lámina de vidrio opaco: La lámina de vidrio es utilizada para difuminar el contenido de las celdas del tablero y así visualizar mejor las luces de los leds, además de que este es un vidrio antireflejo, lo que hace al tablero cómodo de ver incluso en espacios con luces.

Arduino NANO: Esta es una placa pequeña basada en el microcontrolador ATmega328, y es utilizada en este proyecto para ejecutar el código y hacer funcionar correctamente a los distintos componentes del tablero.
